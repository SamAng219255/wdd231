About Me

Hi, I'm Sam Anguiano. I'm from up in Washington state and I've always enjoyed logic and problem-solving. Programming and math have been big parts of my life for as long as I can remember, partly because I've always enjoyed them, and partly because I grew up in a very nerdy family. My dad works in software, so I started learning to code relatively soon, and since then I've had the chance to experiment with a variety of languages and projects. Along the way, I've built my own website, helped on a project developing a game based on a webcomic I enjoyed, and even had an app published on the Apple App Store for a time.

I love projects that let me learn or solve tricky problems. Whether that's solving interesting math, debugging stubborn code, or looking for creative and clever solutions. I've done some fun projects including ones I've mentioned above, a C program that generates high quality aerial maps of Minecraft worlds (quickly and efficiently, which is no mean feat as it turns out), and creating a website to host the 3rd party tabletop rpg content I've written.

Outside of programming, I enjoy solving interesting math, playing table-top rpgs with friends, playing a handful of games (Minecraft and World of Warcraft seem to be my "go to"s these days), reading novels, and watching some animes.
