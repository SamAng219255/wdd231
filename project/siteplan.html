Site Name
Mordan's Tools
It is important to note that this is "a" Pathfinder website and not "the" Pathfinder website so I can't name it after Pathfinder. It also isn't tied to a location or organization with a name to reference. As such, I am using one of the common naming conventions for this kind of site, which is also used by the reference document site that was adopted as the official site. The site is named in reference to a recognizable and relevant character from my own official campaign setting. This makes the name easily memorable, relevant to the content, recognizable by anyone that has seen my other content, and avoids plastering my own name across the site because I don't like having my name plastered everywhere. The site is then referred to as the character's "tools" since the purpose of the site is to provide relevant tools as detailed below.
Site Purpose
The site gives a summary of what Pathfinder is, links to where one can find additional resources, a sign-up for finding groups to play with, and a tool for tracking rolling groups of dice (since Pathfinder often requires rolling multiple dice od varying size), all to the end of easing access to play.
Scenarios
  • I'm brand‑new to tabletop RPGs, what exactly is Pathfinder and how does it differ from D&D?
  • How can I find people to play with?
Color Schema
Four distinct themes:

Theme 1 (Light Sand)

General theme. Used for the background and for most elements.

Theme 2 (Dark Sand)

Used for the header, footer, and some elements set against the background.

Theme 3 (Leather)

Used for a small number of elements that require a greater degree of attention.
Typography
All text will be in this font. The site uses "Helvetica" font with fall-backs to "Arial" and to the browser's generic sans-serif font. Additional letter spacing of 0.01em is also added to make the text more readable.
